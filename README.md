## Getting Started

To get started, you need to go through a few steps:

### Installation

2. Clone the repo
```sh
git clone [https://github.com/github_username/repo_name.git](https://github.com/chainxvi/funding-nextjs.git)
```
3. Install `node_modules`
```sh
npm install
   
or
   
yarn install
```
4. Run the project
```sh
npm run dev
  
or
  
yarn dev
```
